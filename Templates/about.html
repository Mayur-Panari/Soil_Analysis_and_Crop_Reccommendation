Project Mission

Our mission is to empower farmers with real-time soil analysis and accurate crop recommendations using machine learning models, ensuring sustainable agriculture and high-yield crops.

Machine Learning Model

The model uses historical data, such as soil nutrients, pH, and rainfall, to provide accurate recommendations for the best crop choices for each farmer's soil conditions.

Accuracy of the Model

We use Logistic Regression for the finest accuracy, achieving over 96% in crop recommendation through continuous learning from new data.
